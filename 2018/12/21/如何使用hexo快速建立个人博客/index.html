如何使用hexo快速建立个人博客

Posted on 2018-12-21

一个如何使用hexo快速建立个人博客的简单教程

简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装

因为是基于nodejs开发的所以肯定先要安装的，还有需要git

mac用户假如是安装过Xcode，git就已经安装好了

nodejs更简单，直接去它官网上下载。

安装 Hexo

所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。

打开terminal

建站

安装好了之后 terminal里cd 一个用来工作的文件夹

初始化

进入新建的projectname并且安装需要的nodejs的库

新建完成后，指定文件夹的目录如下：

_config.yml里面是

网站的 配置 信息，您可以在此配置大部分的参数。

写文章

在工作目录下

然后source文件夹里面的post里就是新建的文章文件

格式是markdown，往里面写完文章保存起来

运行博客

默认是在4000端口

这是不要关terminal，打开浏览器，输入http://localhost:4000/

就可以看到你的博客了。
